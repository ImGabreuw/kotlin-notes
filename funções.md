# Funções

### Como usar?

  ```kotlin
  fun max(a: Int, b: Int): Int {
    return if (a > b) a else b
  }
  ```
  
* 1º - **fun** = palavra reservada para indicar uma função
* 2º - nome da função (**IMPORTANTE:** utilize um nome que descreva o que essa função faz)
* 3º - parâmetros (**OBS:** são colocados entre parênteses)
* 4º - se necessário, defina o tipo de retorno
* 4º - abrir e fechar os colchetes
* 5º - no corpo da função codifique o que a função irá fazer
