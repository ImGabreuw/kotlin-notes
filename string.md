# String

## Propriedades

### ```isEmpty()```

* O que é?
  
  Se não conter nenhum caractere na string, o método retorna ```true```, caso contrário, ```false```
  
* Como usar?
  
  ```kotlin
  var language = "Kotlin"
  println(language.isEmpty()) // false

  language = ""
  println(language.isEmpty()) // true
  ```
